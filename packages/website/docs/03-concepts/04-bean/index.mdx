## Overview

In this section, you will understand what is Bean and how to use it.

## What is Bean?

Bean is an object that is managed and constructed by Clawject container it can have dependencies, and it can be a dependency for other Beans.
Almost anything can be a Bean, but there are some restrictions:
  - Should not be `undefined` or `null`
  - Should not have type `never`
