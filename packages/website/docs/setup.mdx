## Prerequisites
  - Node.js version 16 or above
  - TypeScript version 4.8 - 4.9

## Installation

```bash
npm install clawject
# or
yarn add clawject
```

<hr />

:::caution Warning

If you're going to use **ts-loader** - make sure **transpileOnly** mode is disabled.
Basically, **transpileOnly** disables the ability to perform static type checking, which is required for **Clawject** to work.

[ts-loader#transpileOnly](https://github.com/TypeStrong/ts-loader#transpileonly)

:::

### Webpack and ts-loader

:::caution Warning

Currently, **ts-loader** has an issue that breaks **hot-reloading** for **Clawject**, so if you
want hot-reloading works correctly - please check [Webpack, ts-loader and ttypescript](#webpack-ts-loader-and-ttypescript).

[GitHub issue](https://github.com/TypeStrong/ts-loader/issues/1615)

:::

To start using Clawject with **webpack and ts-loader**, you need to pass typescript transformer.

```ts title="TypeScript based webpack configuration"
import ClawjectWebpackPlugin from 'clawject/plugins/webpack';
import ClawjectTypescriptTransformer from 'clawject/transformers/typescript';

export default {
    module: {
        rules: [
            {
                test: /\.ts$/,
                loader: 'ts-loader',
                options: {
                    getCustomTransformers: () => ({
                        before: [ClawjectTypescriptTransformer()]
                    })
                }
            }
        ],
    },
    plugins: [
        new ClawjectWebpackPlugin()
    ]
};
```

```ts title="JavaScript based webpack configuration"
const ClawjectWebpackPlugin = require('clawject/plugins/webpack').default;

export default {
    module: {
        rules: [
            {
                test: /\.ts$/,
                loader: 'ts-loader',
                options: {
                    configFile: 'tsconfig.json'
                }
            }
        ],
    },
    plugins: [
        new ClawjectWebpackPlugin()
    ]
};
```
