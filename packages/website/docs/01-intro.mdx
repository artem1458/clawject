## What is Clawject
Clawject is a Dependency Injection framework for TypeScript.

## Why Clawject
Clawject allows you to not write any boilerplate code that's connecting your classes,
use <b>TypeScript type system</b> to define what you need âœ¨

```ts title="main.ts"
import { Bean, CatContext, ContainerManager } from 'clawject';

interface IBar {}
class Bar implements IBar { /* ... */ }
class Foo {
    constructor(
        // highlight-next-line-green
        private bar: IBar // <-- Clawject will automatically inject implementation of IBar interface
    ) {}
    /* ... */
 }

class MyContext extends CatContext {
    foo = Bean(Foo);
    bar = Bean(Bar);
}

ContainerManager.init(MyContext);
```
